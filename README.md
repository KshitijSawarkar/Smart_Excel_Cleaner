# Smart_Excel_Cleaner
Web app to clean and preprocess messy Excel files automatically
📊 Excel Cleaner Web App

A web application that automatically cleans messy Excel files by removing duplicates, handling missing values, and formatting data.

🚀 Features

- Upload Excel files (.xlsx, .csv)
- Automatic data cleaning
- Remove duplicates
- Handle missing values
- Format columns
- Download cleaned file

🛠 Tech Stack

- Frontend: (React / HTML / CSS)
- Backend: (Node.js / Python / Flask)
- Libraries: Pandas / OpenPyXL

🎯 Use Case

Saves hours of manual Excel cleaning for businesses, students, and analysts.

🔥 Future Improvements

- AI-based cleaning suggestions
- Dashboard analytics
- Cloud storage integration
